Coloca aqui los archivos de idioma de Tesseract.

Requerido para OCR en C#:
- spa.traineddata (recomendado)
- eng.traineddata (opcional como respaldo)

Puedes descargar los .traineddata oficiales desde:
https://github.com/tesseract-ocr/tessdata

Luego define (opcional) la variable de entorno TESSDATA_PATH
apuntando a esta carpeta si usas otra ubicacion.
